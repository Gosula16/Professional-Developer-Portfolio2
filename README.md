# Professional Developer Portfolio

Personal portfolio website built with React, TypeScript, Vite, and Tailwind CSS.

## Live site

- Hosted URL: [https://gosula16.github.io/Professional-Developer-Portfolio2/](https://gosula16.github.io/Professional-Developer-Portfolio2/)

## Tech stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Local development

```powershell
npm install
npm run dev
```

## Production build

```powershell
npm run build
```

## Deployment

This repo is configured to deploy automatically to GitHub Pages through GitHub Actions on every push to `main`.

Workflow:

- [`.github/workflows/deploy-pages.yml`](D:/Professional-Developer-Portfolio2/.github/workflows/deploy-pages.yml)
